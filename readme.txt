Professional PDF viewer with Mozilla PDF.js and SEO optimization features.

== Description ==

PDF Embed & SEO Optimize is a comprehensive WordPress plugin that allows you to embed PDF documents directly into your posts and pages using Mozilla's PDF.js viewer. The plugin includes advanced SEO optimization features to ensure your PDF content is properly indexed by search engines.

**Key Features:**

* Mozilla PDF.js integration for reliable PDF rendering
* SEO-optimized PDF embedding
* Responsive design that works on all devices
* Customizable viewer controls
* Support for password-protected PDFs
* Fast loading with lazy loading support
* Accessibility compliant
* Translation ready

**Usage:**

Use the shortcode `[pdf_viewer url="path/to/your/file.pdf"]` to embed any PDF document.

**Shortcode Parameters:**

* `url` - PDF file URL (required)
* `width` - Viewer width (default: 100%)
* `height` - Viewer height (default: 600px)
* `title` - PDF title for accessibility
* `download` - Enable download button (true/false)
* `print` - Enable print button (true/false)
* `search` - Enable search functionality (true/false)

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/pdf-embed-seo-optimize/`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the shortcode in your posts or pages

== Frequently Asked Questions ==

= Does this work with all PDF files? =

Yes, the plugin works with all standard PDF files and supports password-protected documents.

= Is the plugin mobile-friendly? =

The viewer is fully responsive and optimized for mobile devices.

= Does it affect my site's loading speed? =

No, the plugin uses lazy loading and efficient caching to ensure optimal performance.
